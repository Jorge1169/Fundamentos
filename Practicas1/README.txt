# Java Fundamentos

Este proyecto es un programa sencillo en Java que consiste en un menú interactivo.

## ¿Qué hace?

El programa permite:

- Sumar dos números
- Convertir texto a mayúsculas
- Guardar calificaciones
- Calcular el promedio
- Salir del programa

## Temas que se practican

- Uso de Scanner
- Métodos
- Switch
- Arrays
- Ciclos do while
- Variables y operaciones básicas

## Ejemplo del menú

=== MENU ===
1. Calcular suma
2. Convertir texto a mayusculas
3. Capturar calificaciones
4. Mostrar promedio
0. Salir

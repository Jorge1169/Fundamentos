# javaFundamentos2

Proyecto Maven para importar en NetBeans.

Todas las clases están en el paquete:

`com.curso.refactor`

## Objetivo

Este proyecto es un ejercicio integrador de refactorización para validar conocimientos de los módulos 1 al 5.

El código **compila y ejecuta**, pero tiene duplicación intencional y oportunidades de mejora para que el alumno refactorice a nivel de líneas de código.

## Conceptos incluidos

- Módulo 1: variables, operadores, control de flujo, métodos y consola.
- Módulo 2: clases, objetos, encapsulamiento, herencia, interfaces y polimorfismo.
- Módulo 3: `List`, `Set`, `Map`, iteración y Streams.
- Módulo 4: excepciones personalizadas, manejo de errores, Clean Code y SOLID.
- Módulo 5: lambdas, interfaces funcionales, `Optional`, `java.time` y programación funcional.

## Reto

Refactorizar el proyecto para:

1. Extraer validaciones repetidas.
2. Reemplazar strings mágicos por enums.
3. Usar `Optional` en búsquedas.
4. Usar Streams para duplicados, búsquedas y reportes.
5. Centralizar `try-catch` del `Main` usando lambdas.
6. Mantener la herencia e interfaces.
7. Evitar métodos largos.
8. Eliminar código duplicado en registrar y modificar.
